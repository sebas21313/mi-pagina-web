Bienvenido a mi página Web

¿Qué es PHP?

  • Es un Lenguaje de Progrmación del lado servidor. usando principalmente para crear páginas web dinámicas
  • Se integra fácilmente con HTML, CSS y JavaScript.
  • Es gratuito, de código abierto y muy utulizado rn sitios web, junto con base de datos como MYSQL

Caraterísticas principales

  • Lenguaje Interpretado (se ejecuta en el servidor)
  • Compatible con la mayoría de servidores Web (Apache, Nginx, IIS)
  • Mitiplataformas (Windows, Linux, macOS)
  • Se conbina para base de datos CRUD (Crear, Leer, Actualizar, Eliminar).
  • Muy usuado en CMS como WordPress, Joomla, Moodle.
